# TESLO API

1. Clone the project

2. `pnpm install`

3. Copy the file `.env.template` rename to `.env`

4. Change environment variables

5. Up database in docker

```shell
docker-compose up -d
```

6. Execute seed

```
/api/seed
```

7. Run server

```bash
pnpm start:dev
```
